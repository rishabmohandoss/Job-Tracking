# Job Tracking

A beautiful, simple career tracking website to monitor job applications across companies.

## Features

- 🔐 Password protected (09132006)
- 📱 Dark theme with Apple-style design
- 🏢 Track companies and career page links
- ✅ Mark applications as complete with position tracking
- 📝 Add detailed notes per company (recruiter info, what they're looking for, people contacts)
- 📎 Upload files (resume, cover letter, etc.)
- 💾 Persistent local storage (saves automatically)
- 📤 Export/Import data for backup and cross-device sync
- 🌐 Works offline

## Quick Start

1. Open `index.html` in your browser
2. Enter password: `09132006`
3. Start adding companies and tracking your applications

## How to Deploy to GitHub Pages

1. Go to repository Settings → Pages
2. Under "Build and deployment":
   - Source: Deploy from a branch
   - Branch: main, / (root)
   - Click Save
3. Your site will be live at: `https://rishabmohandoss.github.io/Job-Tracking`

## Using the App

- **Add Companies:** Enter company name and career page URL
- **Track Applications:** Check "Applied" and select positions
- **View Details:** Click company name to see detail page
- **Add Notes:** Include recruiter info, requirements, people contacts
- **Upload Files:** Add resume, cover letter, or other relevant files
- **Backup Data:** Click "Export" to download your data; use "Import" to restore

## Data Storage

- All data is stored locally in your browser
- No data is sent to any server
- Use Export/Import to sync between devices
- Regularly export your data as backup
